# Bem vindo ao repositório do Ignite proporcionado pela Rocketseat

## O que é o Ignite?
O Ignite é o Bootcamp antigo da rocketseat porém separado por trilhas individuais: React, NodeJS e React Native (incoming)
  
## Para quem é?
Se você não tem nenhum conhecimento prévio com programação, esse treinamento não é para você, o foco é trazer imersão para aqueles que já dominam uma base mais sólida de programação
